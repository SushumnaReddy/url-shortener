# 🚀 URL Shortener

A production-oriented **Full Stack URL Shortener** built using **React, FastAPI, PostgreSQL, SQLAlchemy, and Docker**. The application allows users to create custom short URLs, generate QR codes, track analytics, and set expiration dates for links.

---

## 🌟 Features

* 🔗 Shorten long URLs instantly
* 🏷️ Custom aliases for branded links
* 📊 Click analytics
* 📱 QR code generation
* ⏳ URL expiration support
* 📋 One-click copy to clipboard
* ❤️ Health check API
* 🎨 Modern responsive React UI
* ⚡ FastAPI REST APIs
* 🐘 PostgreSQL database
* 🐳 Docker support
